# Burbank Roofing Website

A professional roofing company website for Burbank Roofing.

## Structure

```
Burbank Roofing Website/
├── index.html          # Home page
├── about.html          # About us page
├── faqs.html           # Frequently Asked Questions
├── gallery.html        # Photo gallery
├── reviews.html        # Customer reviews
├── contact.html        # Contact form
├── free-inspection.html # Free inspection request form
├── css/
│   └── styles.css      # Main stylesheet
├── js/
│   └── main.js         # JavaScript functionality
├── images/
│   └── gallery/        # Add project photos here
└── README.md           # This file
```

## Features

- Responsive design (mobile-friendly)
- Modern, professional appearance
- Contact forms with validation
- FAQ accordion
- Photo gallery
- Customer reviews
- Mobile navigation menu

## How to Use

1. Open `index.html` in a web browser to view the website
2. Add your own project photos to the `images/gallery` folder
3. Update phone numbers as needed in all HTML files
4. Customize content to match your business

## Customization

### To change colors:
Edit the CSS variables at the top of `css/styles.css`:
```css
:root {
    --primary-color: #1e3a5f;
    --secondary-color: #c9302c;
    --accent-color: #f39c12;
}
```

### To add gallery images:
1. Add your images to the `images/gallery` folder
2. Update `gallery.html` to reference your images

### To update contact information:
Search and replace phone numbers and addresses across all HTML files.
